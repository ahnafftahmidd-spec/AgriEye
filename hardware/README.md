# Hardware Overview

This folder contains all hardware-related information for the AgriEye UAV.

## Core Components
- Raspberry Pi (Flight Controller)
- Brushless DC Motor (Pusher Configuration)
- ESC (30A)
- LiPo Battery (3S)
- Servos (Aileron, Elevator, Rudder)
- RC Receiver & Transmitter
- IMU Sensor
- GPS Module
